

A postgres database dump of over 3 mil records of indian news articles [from the hindu and times of india] and the permalink to that particular article.

If you use this dataset, consider giving a star ‚≠ê

# Schema

```
news (
    id bigint,
    title text,
    published_date date,
    news_provider text,
    link text
)
```
